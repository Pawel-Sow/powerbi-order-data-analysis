Order Data Analysis – Power BI Dashboard

This is my second data analysis project based on the Superstore dataset from Kaggle.
The goal of this project was to analyze product sales and build a simple analytical dashboard.

Project Workflow

The original dataset was imported directly into Microsoft Power BI.
Using Power BI transformations and the DAX language, the data was reshaped into a star schema.
The final step was data visualization, where an interactive dashboard was created.

Tools Used
Power BI
DAX
